### Who we are

A small group of tech friends based in Indonesia. No company, no org chart — just people who happen to enjoy the same things and end up collaborating anyway.

We mess around with **networking**, **system stuff**, **automation**, and occasionally **AI**. Most of it lives on Linux machines, inside VMs, or behind a MikroTik somewhere.

Some things get shipped. Some things stay half-finished in a repo nobody opens. Either way, something gets learned.

```
Type       →  Casual group / friend circle
Base       →  Indonesia
Vibe       →  Build · Break · Learn · Repeat
Status     →  Always tinkering
```

### The unofficial manifesto

```yaml
we_are_not:
  - a startup
  - looking for investors
  - taking ourselves too seriously

we_do:
  - help each other debug at odd hours
  - share setups, configs, and random findings
  - build things because they're interesting, not because they scale

philosophy: "understand it from the ground up, then build something real"
```
